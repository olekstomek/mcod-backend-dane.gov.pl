A resource is an object that contains the following fields:

  • title (string) - title of the resource
  • description (string) - short description of the resource
  • category (string) - category of the resource
  • openness_score (integer) - openness score of the resource (from 1 to 5)
  • format (string) - resource format (applies when the resource is a file)
  • media_type (string) - type of resource; possible values are: file, website, api
  • file_url (string) - url to the file (applies if the resource is a file)
  • file_size (integer) - file size in bytes (applies if the resource is a file)
  • link (string) - url to the resource
  • views_count (integer) - number of resource views
  • downloads_count (integer) - number of resource downloads
  • created (string) - date of creation of the resource
  • modified (string) - date of last modification

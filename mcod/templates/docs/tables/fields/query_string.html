{% trans 'The "q" param is for search. For a better understanding of how it works, see the examples below.' %}

{% trans "Searching in specific fields" %}

{% trans "The query can run against one or multiple fields." %}
{% trans "Example:" %}
https://api.dane.gov.pl/resources/8162/data?q=col3:Nowak AND col1:Mazowieckie
https://api.dane.gov.pl/resources/8162/data?q=(col3:Nowak OR col3:Kowalski) AND col1:Mazowieckie

{% trans "Searching in all fields" %}

The query can also run against all text-based fields. {% trans "Example:" %}
https://api.dane.gov.pl/resources/8162/data?q=Kowalski AND Mazowieckie

{% trans "Wildcards" %}

Wildcard searches can be run on individual terms, using ? to replace a single character, and * to replace zero or more characters.

{% trans "Example:" %}

https://api.dane.gov.pl/resources/8162/data?q=n?wa* https://api.dane.gov.pl/resources/8162/data?q=col1:dolno*

{% trans "Regular expressions" %}

Regular expression patterns can be embedded in the query string by wrapping them in forward-slashes ("/").

{% trans "Example:" %}

https://api.dane.gov.pl/resources/8162/data?q=/kow[eai]lski/

{% trans "Fuzziness" %}

We can search for terms that are similar to, but not exactly like our search terms, using the “fuzzy” operator.

{% trans "Example:" %}

https://api.dane.gov.pl/resources/8162/data?q=nowk~ https://api.dane.gov.pl/resources/8162/data??q=nawakoki~

This uses the Damerau-Levenshtein distance to find all terms with a maximum of two changes, where a change is the insertion, deletion or substitution of a single character, or transposition of two adjacent characters.

The default edit distance is 2, but an edit distance of 1 should be sufficient to catch 80% of all human misspellings. It can be specified as in following example:

https://api.dane.gov.pl/resources/8162/data?q=nowakoki~1

{% trans "Proximity searches" %}

Proximity query allows the specified words to be further apart or in a different order.

In the same way that fuzzy queries can specify a maximum edit distance for characters in a word, a proximity search allows us to specify a maximum edit distance of words in a phrase.

https://api.dane.gov.pl/resources/7010/data?q="beskidzki dworzec"~5
